Project Overview

Alitasha Lodge Management System is a full-stack web application designed to digitise and streamline lodge operations. It replaces manual booking logs and spreadsheets with a centralised dashboard accessible from any device — phone, tablet, or computer. The system handles reservations, guest management, room allocation, visitor tracking, and business analytics, all in real time.
